TestMan

Developed by Srinivasan and Surya Prakash

TestMan is a desktop utility that automatically captures screenshots and inserts them directly into Microsoft Word documents. Designed for software testing, documentation, QA reporting, training manuals, and project evidence collection.

  • 📸 Capture screenshots instantly using C + V
  • 🗑 Delete the last screenshot using D + V
  • 📄 Automatically inserts screenshots into Word documents
  • 📏 Screenshots fit inside A4 page size
  • ⚡ Lightweight and easy to use
  • 🔒 No screenshots stored permanently on disk
